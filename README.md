# Hospital Emergency Room Dashboard (Excel)

## About the Project
This project focuses on analyzing Emergency Room (ER) patient data using an interactive Excel dashboard.  
The goal of this project is to understand hospital operational performance, patient flow, waiting time, and service efficiency.

The dashboard allows users to analyze key emergency room metrics such as patient count, waiting time, patient satisfaction, admission rates, and demographic distribution.

A slicer is used to filter the data by month. In the dashboard, January is selected for analysis, but the dashboard also allows analysis for other months dynamically.

This project demonstrates how Excel can be used as a data analytics tool for healthcare data visualization and operational insights.

---

## Business Questions
The dashboard was created to answer the following analytical questions:

1. What is the total number of patients visiting the emergency room?
2. What is the average patient waiting time?
3. What is the average patient satisfaction score?
4. What percentage of patients are admitted vs not admitted?
5. How are patients distributed based on department referrals?
6. What is the patient distribution across different age groups?
7. What is the patient attendance status (Delayed vs On-Time)?
8. What is the gender-wise distribution of patients?
9. How do these metrics change when analyzing different months?

---

## Process
The following steps were followed to build the dashboard:

1. **Data Collection**
   - Emergency room patient dataset containing information about wait time, patient satisfaction, admission status, age, gender, and department referrals.

2. **Data Cleaning**
   - Organized the dataset in Excel.
   - Checked for missing values and standardized fields.

3. **Data Analysis**
   - Used Pivot Tables to summarize patient data.
   - Calculated key metrics such as total patients, average waiting time, and satisfaction scores.

4. **Data Visualization**
   - Created charts and KPI cards to display important insights.
   - Added slicers to enable dynamic filtering by month.

5. **Dashboard Creation**
   - Designed an interactive dashboard layout in Excel to present insights clearly.

## Insights
From the dashboard analysis (January selection):

• Total Patients: **513**

• Average Patient Waiting Time: **36.32 minutes**

• Average Patient Satisfaction Score: **4.96**

• Admission Analysis:
  - **52% patients admitted**
  - **48% patients not admitted**

• Department Referral:
  - **299 patients were not referred to any department**
  - Remaining patients were referred to specialized departments such as General Practice, Orthopedics, Physiotherapy, Neurology, and Cardiology.

• Age Group Analysis:
  - The **0–9 and 10–19 age groups show the highest ER visits**.

• Attendance Status:
  - **62% patients experienced delays**
  - **38% patients were attended on time**

• Gender Distribution:
  - **53% Male**
  - **47% Female**

These insights highlight patterns in patient flow and operational efficiency in the emergency room.

## Solutions & Recommendations
Based on the insights, the following improvements could help hospitals enhance ER operations:

**1. Reduce Patient Waiting Time**
- Improve triage prioritization
- Increase staff availability during peak hours
- Implement digital queue management systems

**2. Improve Patient Flow Management**
- Introduce digital check-in systems
- Streamline registration and patient routing

**3. Strengthen Referral Processes**
- Establish clear referral protocols
- Implement decision-support systems for ER staff

**4. Manage Delayed Attendance**
- Use automated appointment reminders
- Optimize scheduling systems

**5. Improve Pediatric Emergency Care**
- Establish dedicated pediatric emergency units due to high visits from younger age groups
